1. Introducción

Este manual describe paso a paso cómo instalar, configurar y poner en funcionamiento el Sistema de Valoración y Calificación del Desempeño de Docentes y Alumnos (SIVACAD). Está dirigido al personal técnico, desarrolladores y soporte responsables de la implementación.

4. Instalación de la Base de Datos

4.1 Crear la base de datos

mysql -u root -p
CREATE DATABASE sivacad_isc CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;

4.2 Importar el esquema

mysql -u root -p sivacad_isc < database/sivacad_isc.sql
for file in database/migrations/*.sql; do
  mysql -u root -p sivacad_isc < "$file"
done

4.3 Verificar

mysql -u root -p -e "USE sivacad_isc; SHOW TABLES;"

Deberían aparecer las 67 tablas del sistema.

5. Configuración del Backend

5.1 Instalar dependencias

cd backend
npm install

5.2 Configurar .env

cp .env.example .env

Edite el archivo .env con los valores correctos:

PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=tu_contraseña
DB_NAME=sivacad_isc
JWT_SECRET=generar_un_secreto_aleatorio_seguro
JWT_EXPIRES_IN=24h
GEMINI_API_KEY=tu_api_key_de_gemini
SMTP_HOST=smtp.office365.com
SMTP_PORT=587
SMTP_USER=correo@tesi.edu.mx
SMTP_PASS=contraseña
FRONTEND_URL=http://localhost:5173
FLASK_ML_URL=http://localhost:5001

5.3 Verificar

node -e "require('./src/app.js'); console.log('Configuración OK');"

10.4 Probar desde móvil

  1. Conecte el móvil a la misma red WiFi que el servidor.
  2. Obtenga la IP local: ipconfig (Windows).
  3. Acceda desde el móvil: http://IP:5173.
  4. El CORS acepta IPs LAN (192.168.x.x, 10.x.x.x, 172.16-31.x.x).

11. Despliegue en Producción

11.1 Compilar frontend

cd frontend
npm run build
# Archivos estáticos en frontend/dist/

11.2 Configurar Nginx

server {
    listen 80;
    server_name sivacad.tesi.edu.mx;
    root /var/www/sivacad/frontend/dist;
    index index.html;

    location /api/ {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location / {
        try_files $uri $uri/ /index.html;
    }
}

11.3 Gestionar con PM2

npm install -g pm2
cd backend && pm2 start src/server.js --name sivacad-backend
cd backend/ml && pm2 start app.py --interpreter python --name sivacad-ml
pm2 save
pm2 startup

11.4 SSL con Certbot

sudo certbot --nginx -d sivacad.tesi.edu.mx

12. Solución de Problemas Comunes

Error: Conexión MySQL

Causa: MySQL no corre o credenciales incorrectas.

sudo systemctl status mysql  # Linux
net start MySQL              # Windows

Error: Module not found

Solución: Reinstalar dependencias:

cd backend && rm -rf node_modules && npm install

Error: CORS desde móvil

Causa: IP no está en lista blanca. Verifique rango LAN.

Error: Gemini API Key

Solución: Configure GEMINI_API_KEY en .env.

Error: Puerto en uso

# Linux
lsof -i :3000 && kill -9 PID
# Windows
netstat -ano | findstr :3000 && taskkill /PID PID /F

Error: Build frontend falla

cd frontend && rm -rf node_modules && npm install && npm run build

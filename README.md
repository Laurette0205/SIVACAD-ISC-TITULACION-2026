## Stack Tecnológico

- **Backend**: Node.js + Express
- **Frontend**: React + Vite
- **Base de datos**: MySQL
- **Machine Learning**: Python (Flask)
- **Generación de PDF**: Puppeteer + Dompdf (PHP)
- **Generación de Excel**: ExcelJS (Node.js) + PhpSpreadsheet (PHP)

## Módulos

- Gestión de alumnos, docentes y grupos
- Inscripciones y reinscripciones
- Kardex y evaluación académica
- Trámites escolares (baja, cambio de escuela/carrera)
- IA de deserción académica
- IA de acompañamiento estudiantil (Bienestar)
- IA de becas
- OCR para actas de calificaciones
- Chatbot y asistente académico
- Reportes y auditoría
- Soporte técnico
